# 📅 Event Management & Registration System

A **full-stack web application** for managing school events, online
registration, attendance tracking, certificate generation, and
analytics reporting.

## ✨ Features

| Module | Description |
|--------|-------------|
| 🔐 **Authentication** | Login/Register with JWT tokens |
| 📅 **Event Management** | Create, edit, delete events (Admin) |
| 📝 **Online Registration** | Students register with duplicate prevention |
| ✅ **Attendance Tracking** | Manual marking + QR code scanning |
| 🏆 **Certificate Generation** | Auto-generate PDF certificates |
| 📊 **Analytics Dashboard** | Charts, reports, event statistics |

---

## 🛠️ Tech Stack

- **Frontend:** HTML5, CSS3, Vanilla JavaScript, Chart.js
- **Backend:** Node.js, Express.js
- **Database:** MySQL
- **Authentication:** JWT (JSON Web Tokens)
- **PDF Generation:** PDFKit
- **QR Codes:** qrcode library
